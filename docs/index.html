Over het project

Deze repository onderzoekt A003558 als “kritische trapfunctie” voor faseovergangen in informatie-dichtheid, met koppelingen via octonion-algebra en een octahedronale matrix. De renders (uit Blender) illustreren horizonlagen (2^n) en orbits van de verdubbelingsmap.

  • Code & tests: Python package met pytest (alle tests groen ✅)
  • Visualisaties: Blender scenes (statisch) + scriptable pipeline
  • Data: OEIS A003558, back-front orbits, octonion-consistentie
