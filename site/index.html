A003558 — Universal Clock

Testbare hypothese: verdubbelings-fractalen, horizons 2^n, en octonionale koppeling.

Bekijk op GitHub Galerij

Over het project

Deze repository onderzoekt of A003558 als “kritische trapfunctie” kan fungeren voor faseovergangen in informatie-dichtheid, met koppelingen via octonion-algebra en een octahedronale matrix. De Blender-renders illustreren horizonlagen (2^n) en orbits van de verdubbelingsmap.

  • Code & tests: Python-package met pytest (alle tests groen ✅)
  • Visualisaties: Blender-scenes (statisch) + scriptable pipeline
  • Data: OEIS A003558, back-front orbits, octonion-consistentie

Hoe reproduceer je?

  1. Clone de repo en activeer venv → pytest moet groen zijn.
  2. Open Blender, render hoge-res beelden naar pics/ (bijv. 01.webp … 05.webp).
  3. Werk deze galerij bij door kaartjes toe te voegen/vervangen.
